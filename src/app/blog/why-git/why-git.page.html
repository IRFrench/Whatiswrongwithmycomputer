Why we use Git Overview Why do you think you need git? If you are just looking to save some files to the cloud, There may be some better options out there. This blog is not just going to address the basics of git, but also the amazing features that it has, including github pages, gitlab CI/CD and others.
 The basics Starting with the basics, git has a very easy to use system. Using repositories to create folders of files and save them to the cloud. This allows for another copy of the files to be saved which should be reason enough to use git.
